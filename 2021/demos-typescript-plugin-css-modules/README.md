## 步骤

```
➜ npx create-react-app project-name --template typescript
➜ yarn add -D typescript-plugin-css-modules
```

`tsconfig.json`

```
    "plugins": [
      {
        "name": "typescript-plugin-css-modules"
      }
    ]
```
